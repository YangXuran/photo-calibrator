# Film Scan Auto-Level/Crop — Implementation Plan

> **Agent role:** Agent C per AGENTS.md §15.5
> **Goal:** Detect film borders, estimate rotation, output crop box. Core layer, pure functions, no disk I/O.
> **Tech:** NumPy, OpenCV (Canny + HoughLinesP + quad fitting)

---

## Architecture

```
detect_film_frame(img_rgb) → FilmScanResult
  ├─ _find_edge_lines()     → Canny + HoughLinesP → dominant line groups
  ├─ _fit_border_quad()     → group lines → 4 corners → confidence
  └─ _compute_crop()        → from corners → angle + crop rect
```

## Data Model

```python
@dataclass(frozen=True)
class FilmScanResult:
    angle_deg: float          # rotation angle (positive = clockwise from horizontal)
    corners: list[tuple]      # 4 corner points [(x,y), ...] of detected frame
    crop_x: int               # crop rect top-left x
    crop_y: int
    crop_w: int
    crop_h: int
    confidence: float         # 0.0-1.0, low = skip / use original
    border_type: str          # "black", "white", "mixed", "unknown"
```

## Algorithm

1. **Grayscale + Canny**: low/high thresholds adaptive from image stats
2. **HoughLinesP**: probabilistic, tuned for long lines (film borders)
3. **Line grouping**: cluster by angle (0°, 90° for level; rotated for tilted)
4. **Quad fitting**: find intersections of line groups → 4 corners that form a convex quadrilateral
5. **Angle**: from dominant line orientation
6. **Crop**: largest inscribed rectangle of detected quad
7. **Confidence**: drops if < 4 clean lines found, or quad is implausible (too small, warped)

## Edge Cases

- **No border detected**: return confidence=0, crop_rect=full image
- **Uneven lighting**: Canny adaptive thresholds handle
- **Sprocket holes**: small holes filtered by Hough line min length
- **Very small rotation (< 0.5°)**: angle rounded to 0
